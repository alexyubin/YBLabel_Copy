# YBLabel_Copy

## 一个属性 给UILabel添加长按复制


## demo
![image](https://raw.githubusercontent.com/alexyubin/YBLabel_Copy/master/Demo.gif)


## Installation
> **手动拖入**
> 将 YBLabel_Copy 文件夹拽入项目中，导入头文件：#import "#import "UILabel+Copy.h"


## How to use
<pre><code>
UILabel *label= [[UILabel alloc] init];
//使UILabel拥有长按复制功能
label.copyingEnabled = YES;
</code></pre>
